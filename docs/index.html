Instructions

Create a spreadsheet with the columns: Artist, Title, Date, Medium, Price, and QR. Use that capitalization and spelling. Order does not matter, just make sure that the column headers are in the first row.

Export your spreadsheet as a TSV (tab separated value) file and drag and drop it onto this page to generate your tags.

Example:

Artist Title Date Medium Price QR
Sample Artist Sample Title 2025 Digital $100 https://example.com

Tags will appear different when printed.

Ctr-P, Apple-P or File > Print and save as PDF
